## 🚀 Começando

```bash
# Clone este repositório
git clone https://github.com/KerubinDev/MediFlow.git

# Navegue até o diretório
cd MediFlow

# Instale as dependências
pip install -r requirements.txt

# Inicie o servidor
python run.py
```
